Spend Finance is a personal finance tracking web application that allows users to record, manage, and analyze their income and expenses in a structured and visual way.

The system features a dashboard with real-time data visualization, helping users understand their financial 
trends through charts and analytics. Users can also view and filter transaction data in an interactive table,
making it easier to track spending and income patterns.

a. Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python
- Database: MySQL
- Visualization: JavaScript (Chart.js )
- Chatbot : python ( TF-idf and consine similarity)

b. Key Features
- Dashboard overview of financial data
- Monthly income and expense summary
- Interactive charts for analytics
- Transaction management system
- Filtering and sorting transactions
- Clean dark UI
- chatbot for asking the question about investing
